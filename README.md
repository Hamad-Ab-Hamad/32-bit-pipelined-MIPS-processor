# 32-bit-pipelined-MIPS-processor
This is a project report about designing and implementing a custom 32-bit RISC processor, a stripped-down MIPS processor. The objective is to build a 32-bit version of the MIPS processor with a minimal instruction set from the MIPS ISA while implementing the pipeline datapath version of the processor utilizing the VHDL hardware description language. The goal of this project is to get a more practical, hands-on approach to computer architecture design problems. All code and simulations are done in the Vivado software
